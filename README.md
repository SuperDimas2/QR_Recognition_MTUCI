# QR Recognition Bot
Telegram Bot for QR-code recognition and reading implied text.
# Requirements
- Torch
- PyTelegramBotAPI
- YOLO
# How to Launch
Insert your Telegram Bot token from @BotFather into qrbot.py

Launch qrbot.py

Send a picture to bot
